JavaScript 개요

  • 자바스크립트는 웹 브라우저에서 동작하는 프로그래밍 언어 이다.
  • 종류
    • ECMAScript : 자바스크립트의 공식적 이름 (JavaScript Versions)
    • 클라이언트 측 자바스크립트 : 웹브라우저에서 동작되는 자바스크립트로 브라우저에 내장되어있다.(ECMAScript(중심언어) + 브라우저 API)
    • 서버 측 자바스크립트 : 웹서버에서 동작되는 자바스크립트로 서버에 설치하여 사용한다.(Node.js 등)
  • 자바스크립트는 페이지를 동적으로 표현할 수 있게 한다.
    • ???
  • 실행방법
    • html문서에 삽입하여 사용하는 방법
      • 이벤트 속성에 작성
        • < input type = "button" onclick = "javascript코드" >
      • < script >태그 사용
        • < script > javascript코드 < /script >
      • 외부 js파일(*.js)과 연결
        • < script src="파일경로" > < /script >
    • 개발자도구의 웹브라우저 콘솔(개발자 도구)에서 실행하는 방법.

기본 규칙

  • 대문자와 소문자를 구별한다.
  • 문장의 끝에 세미콜론(;)을 반드시 붙인다.
  • 주석문은 /* 문장 */ 또는 // 으로 만든다.
  • 여러 문장은 { }으로 감싼다.

출력방법

  • 브라우저 console에 출력
    • console.log("hello javascript")
  • 경고창에 출력
    • alert("hello javascript")
  • html 태그에 출력
    • document.write("hello javascript")
    • document.getElementById('text').innerHTML = 'hello'
    • 예
      • <!DOCTYPE html>
      <html>
      <body>
      <h2>My First JavaScript</h2>
      <button type="button" onclick="document.getElementById('demo').innerHTML = Date()">
      Click me to display Date and Time.</button>
      <p id="demo"></p>
      </body>
      </html>
