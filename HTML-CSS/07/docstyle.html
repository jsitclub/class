화면배치

  • 요소배치
    • position : 요소배치 ( 예제 )
      • static : 일반적 배치 (상->하 , 좌->우)
      • relative : 원래 있어야 하는곳에서 지정된 위치만큼 이동
      • absolute
      • fixed
      • redlative,absolute,fixed 는 left,height와 함께 사용하여야 함
      • 예)
        • p{
          • position:relative;
          • left:100px;
          • height:100px;
        • }
    • float : 좌우배치 ( 예제 )
      • left
      • right
      • none
    • clear : 좌우배치해제
      • left
      • right
      • none
    • display : 배치방식 변경
      • block
      • inline
      • none
  • 배경설정
    • background : 배경색
    • background-image : 배경이미지
      • url("이미지 경로")
    • background-repeat : 배경이미지 반복
      • repeat
      • repeat-x
      • repeat-y
      • no-repeat
    • background-attachment : 배경이미지 고정여부
      • scroll
      • fixed
    • background-position : 배경이미지 위치
        • top left top center top right
        center left center center center right
        bottom left bottom center bottom right
