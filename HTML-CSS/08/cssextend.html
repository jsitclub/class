미디어쿼리 (예제)

  • 하나의 웹페이지를 상황에 따라 스타일로 다르게 보여준다
    미디어 쿼리는 viewport의 높이/너비, 디바이스의 높이/너비, 타블렛이나 폰의 가로/세로 등을 체크할수 있다.
  • 적용
    • @media로 CSS에 적용
        • <style>
        @media all and (max-width : 480px)
        {
        div {
        color:skyblue;
        }
        }
        </style>
    • <style> 태그의 media 속성으로 적용
        • <style type="text/css" media="all and (max-width : 480px)">
        div {
        color:skyblue;
        }
        </style>
    • <link> 태그를 이용하여 외부 CSS파일에 적용
        • <link type="text/css" media="all and (max-width : 480px) href="main_style.css >
  • 미디어 타입
    • all
    • screen
    • print
  • 조건적용
    • 웹페이지의 너비 : width, min-width, max-width
    • 웹페이지의 높이 : height, min-height, max-height
    • 디바이스의 너비 : device-width, min-device-width, max-device-width
    • 디바이스의 높이 : device-height, min-device-height, max-device-height

셀렉터

  • css에서 요소에 스타일을 적용하는 방법의 패턴들을 말한다.('.', '#', ':' 등)
  • ':' 셀렉터를 이용하여 마우스를 이벤트(올려놓았을때, 클릭했을때, 포커스받았을때)처럼 사용 할 수 있다.(예제)

기타 CSS 효과

  • 브라우저 구분 : 브라우저마다 다를수 있을때 속성앞에 붙여 사용한다.
    • -webkit- : 크롬,사파리
    • -moz- : 파이어폭스
    • -o- : 오페라
    • 예) -o-column-count: 3
  • 컬럼 나누기(예제)
    • column-count : 컬럼 나누기
    • column-width : 컬럼 너비
    • column-gap : 컬럼 여백
    • column-rule : 컬럼 선(두께,스타일,색 순서)
  • 둥근 테두리 (예제)
    • border-radius : top-left,top-right,bottom-right,bottom-left 순으로 값(px,%)을 한번에 지정
    • border-top-left-radius : 위-왼쪽 지정
    • border-top-right-radius : 위-오른쪽 지정
    • border-bottom-right-radius : 아래-오른쪽 지정
    • border-bottom-left-radius : 아래-왼쪽 지정
  • 그림자 (예제)
    • box-shadow : 그림자 적용
      • h-shadow : 가로 그림자
      • v-shadow : 세로 그림자
      • blur : 그림자 퍼짐정도
      • spread : 그림자 퍼짐정도
      • color : 색상
      • insert : 안쪽
