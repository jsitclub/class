CSS 기본

  • style 바꾸는 방법
    • 태그의 속성 이용
      • <img src="aa.jpg" width="300">
    • style 속성 이용
      • <p style="color:purple">
    • style 태그 이용
      • 주로 < head> 부분에 작성
      • < style> div{ padding:10px; } < /style>
    • 외부 css파일 이용
      • link 태그를 이용하여 외부에 있는 css파일을 연결
      • <link rel="stylesheet" type="text/css" href="style.css">
    • 우선순위
      • style 속성 > style 태그 > 외부css파일 > 태그의 속성
      • id > class > 태그
  • class 속성 과 id 속성
    • 태그를 구분 하기위해 임의로 붙여준다.
    • class : 그룹 구분값
    • id : 유일한 구분값으로 페이지 안에서 겹치면 안됨
      • <div class="red" id="red01">
        <div class="red" id="red02">
    • style에서 class는 . 으로 id는 # 으로 구분한다.
      • .red{
             color : red;
        }

        #red01{
             padding : 10px;
        }
  • 박스모델
    • content : 내용영역
    • border : 테두리
      • border-width : 두께
        • thin : 얇은두께
        • midium : 중간두께
        • thick : 두꺼운두께
        • 값 : px,pt등으로 설정
      • border-color : 색깔
      • border-style : 테두리 종류
        • solid : 실선
        • dotted : 점선
        • dashed : 점선
        • double : 두줄
        • groove : 입체적-액자
        • ridge : 입체적
        • inset : 입체적
        • outset : 입체적
    • padding : 안여백
    • margin : 바깥여백
    • padding,border,margin은 각각 -top,-right,-bottom,-left 와 조합하여 사용 가능
      예)padding-left, border-top-width, margin-right
