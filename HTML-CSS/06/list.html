목록

  • <ul> : 순서가 없는 목록
    • CSS속성 : list-style-type
      • 예) <ul style="list-style-type:disc">
      • none
      • disc
      • circle
      • square
    • <li> : 항목

  • <ol> : 순서가 있는 목록
    • CSS속성 : list-style-type
      • decimal : 숫자
      • lower-alpha : 알파벳 소문자
      • lower-roman : 로마자 소문자
      • upper-alpha : 알파벳 대문자
      • upper-roman : 로마자 대문자
      • cjk-ideographic : 한자숫자
    • <li> : 항목

  • <dl> : 정의 목록 : 용어사전과 같은 형태
    • <dt> : 항목
    • <dd> : 설명
  • ul, ol, dl 모두 이미지를 구분자로 지정할 수 있다.
