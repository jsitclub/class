  • 웹 페이지에서 데이터를 입력을 받아야 할때는 HTML의 input태그를 이용한다. 하지만 입력된 내용을 처리하는것은 주로 javascript 나 서버 프로그래밍이 필요하다.

< input >

  • < input >태그는 사용자에게 입력을 받을수 있는 태그로,type 속성에 의해 여러모양으로 사용 된다.
  • 속성
    • type : input 종류 설정 (다음표 참조)
    • name : 태그 이름 설정
    • value : 입력값
    • maxlength : 입력가능 길이
    • readOnly : 읽기전용
    • disabled : 활성화여부
    • checked : 선택된 값(체크박스,라디오버튼에서 사용)
      • true / false
        값 모양 설명 예
        사용안함 텍스트 입력 <input value="input" name="input">
        text text 입력 <input type="text" value="text" name="text">
        number 숫자입력 <input type="number" value="number" name="number">
        password 패스워드 입력 <input type="password" value="password" name="password">
        email 이메일 입력 <input type="email" value="email" name="email">
        tel tel입력 <input type="tel" value="tel" name="tel">
        url url입력 <input type="url" value="url" name="url" >
        search search 입력 <input type="search" value="search" name="search" >
        button 버튼 <input type="button" value="button" name="button">
        reset 폼의 입력값들을 초기화 <input type="reset" value="reset" name="reset">
        submit 폼의 입력값들을 전송 <input type="submit" value="submit" name="submit">
        checkbox 독서 코딩 운동 체크박스 <input type="checkbox" value="1" name="checkbox"> 독서
        <input type="checkbox" value="2" name="checkbox" checked="true"> 코딩
        radio 자전거 도보 버스 라디오버튼(옵션버튼) <input type="radio" value="1" name="radio" checked="true"> 자전거
        <input type="radio" value="2" name="radio" > 도보
        range range 입력 <input type="range" value="range" name="range">
        file 파일선택 <input type="file" value="file" name="file" >
        color 색선택 <input type="color" value="color" name="color">
        date 날짜입력 <input type="date" value="2020-04-09" name="date">
        time 시간입력 <input type="time" value="12:00" name="time">
        datetime-local 날짜/시간입력 <input type="datetime-local" value="2020-04-09T12:00" name="datetime-local">
        month 월입력 <input type="month" value="2020-04" name="month" >
        week 주입력 <input type="week" value="2020-W04" name="week" ">
        image submit 버튼의 이미지 정의 <input type="image" src="../resource/img_submit.gif" name="image" >
        hidden 숨겨진 값 처리 <input type="hidden" value="hidden" name="hidden" >

< select >

  • < select >태그는 여러값에서 선택할 있는 태그
  • 속성
    • name : 태그 이름 설정
    • size : 크기(보여줄 행의 갯수)
      • size ="4" : size 미설정 :
    • multiple : 다중선택 여부
      • shift를 누른후 선택
      • 다중선택 설정: 다중선택 미설정 :
    • disabled : 활성화 여부
  • 예
    • <select>
    • <option value="text" > text</option >
    • < option value="number" > number</option >
    • </select >

< textarea >

  • < textarea > 는 여러줄을 입력할수 있는 태그이다.
  • 속성
    • name : 태그 이름 설정
    • rows : 행의 수(세로길이)
    • cols : 열의 수(가로길이)
    • readOnly : 읽기전용
    • disabled : 활성화여부

< button >

  • < button > 은 < input type="button"> 과 모양은 비슷하지만 닫는태그 < /button > 가 필요하고 사이값으로 텍스트나 이미지가 들어갈수 있다는 차이가 있다.
  • 속성
    • type
      • button
      • reset
      • submit
    • value : 버튼의 text
    • name : 버튼의 이름
