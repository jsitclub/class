텍스트 서식

  • <i> <em> : 이탤릭체
  • <b> <strong> : 강조체
  • <u> <ins> : 밑줄
  • <del> : 취소선(가로줄)
  • <sup> <sub> : 위첨자, 아래첨자

    • 목록

  • <ul> : 순서가 없는 목록
    • CSS속성 : list-style-type
      • 예) <ul style="list-style-type:circle;">
      • none
      • disk
      • circle
      • square
  • <li> : 항목

  • <ol> : 순서가 있는 목록
    • 속성
      • 예) <ol type="A" start ="B" reversed>
      • type : 순서의 타입 (1, A, a,I, i)
      • start : 시작번호 (1, A, a,I, i)
      • reversed : 역순여부
  • <li> : 항목

  • <dl> : 정의 목록
  • <dt> : 항목
  • <dd> : 설명
