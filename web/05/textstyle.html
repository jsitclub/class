텍스트 CSS

  • font-family : 한번에 여러개의 글꼴지정
    • 앞의 글꼴이 없으면 순서대로 적용되므로 대표글꼴을 뒤쪽에 적는다
    • font-family : "궁서체","굴림","serif"
    • 대표적 글꼴
      • serif
      • sans-serif
      • cursive
      • fantasy
      • monospace
  • font-size : 글꼴의 크기
    • 크기값 : 크기를 설정 (단위:pt,px,cm,em)
    • inherit(상속) : 주변의(부모의) 텍스트의 크기와 동일
    • 백분율 : 주변의(부모의) 텍스트의 크기와 비례 (100%)
    • font-style : 글꼴의 스타일
      • normal : 일반적 스타일
      • italic : 이텔릭 스타일
      • oblique : oblique 스타일
      • inherit : 주변의(부모의) 텍스트 스타일
    • font-weight : 글꼴의 굵기
      • normal : 일반적 굵기
      • bold : 이텔릭 스타일
      • bolder : oblique 스타일
      • lighter : oblique 스타일
      • inherit : 주변의(부모의) 텍스트 스타일
