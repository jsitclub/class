텍스트관련 CSS ( html5 : CSS 권장 )

  • 태그로 표현한것은 CSS로 표현가능
    • 안녕하세요 jsitclub 입니다.
    • 안녕하세요 jsitclub 입니다.
  • text-decoration : 꾸미기
    • 속성
      • none : 기본값
      • underline : 밑줄
      • overline : 윗줄
      • line-through : 취소선
      • blink : 깜빡이기(파이어폭스,오페라)
      • inherit : 주변속성과 동일
  • color : 글자색
    • 값은 red,RGB(255,0,0),#FF0000과 같이 3가지 형태로 표현가능
  • font-size : 글꼴의 크기
    • 크기값 : 크기를 설정 (단위:pt,px,cm,em)
    • 백분율 : 주변의(부모의) 텍스트의 크기와 비례 (100%)
    • inherit: 주변속성과 동일
  • font-style : 글꼴의 스타일
    • normal : 일반적 스타일
    • italic : 이텔릭 스타일
    • oblique : oblique 스타일
    • inherit: 주변속성과 동일
  • font-weight : 글꼴의 굵기
    • normal : 일반적 굵기
    • bold : 이텔릭 스타일
    • bolder : oblique 스타일
    • lighter : oblique 스타일
    • 숫자값 : (100~900) 100단위로 지정
    • inherit: 주변속성과 동일
  • font-family : 한번에 여러개의 글꼴지정
    • 앞의 글꼴이 없으면 순서대로 적용되므로 대표글꼴을 뒤쪽에 적는다
    • font-family : "궁서체","serif"
    • 대표적 글꼴
      • serif
      • sans-serif
      • cursive
      • fantasy
      • monospace
  • text-shadow : 그림자효과
    • text-shadow : 2px 2px 7px #FF0000
    • 그림자효과를 적용할때는 다음 속성을 순서대로 적용한다.
    • 속성
      • h-shadow : 가로그림자 (단위 : px,pt,cm. 음수도 가능)
      • v-shadow : 세로그림자 (단위 : px,pt,cm. 음수도 가능)
      • blur : 그림자의 퍼짐정도 (단위 : px,pt,cm. 음수도 가능)
      • color : 색상
  • text-align : 정렬
    • 속성
      • left : 왼쪽정렬
      • right : 오른쪽정렬
      • center : 가운데정렬
      • justify : 양쪽정렬
      • inherit: 주변속성과 동일
  • @font-face : 웹폰트
    • 사용자 컴퓨터에 폰트가 없어도 서버의 폰트를 갖고온다.
    • 속성
      • font-family : 사용할 이름을 정한다
      • src : 글꼴파일(ttf,eot)의 경로 예) url("../src/SDMiSaeng.ttf")
      • font-style : 글꼴의 스타일
      • font-weight : 글꼴의 굵기
