텍스트 서식

  • <i> <em> : 이탤릭체
  • <b> <strong> : 강조체
  • <u> <ins> : 밑줄
  • <del> : 취소선(가로줄)
  • <sup> <sub> : 위첨자, 아래첨자

    • 목록

  • <ul> : 순서가 없는 목록
  • <ol> : 순서가 있는 목록
      • 속성
    • type : 순서의 타입 (1, A, a,I, i)
    • start : 시작번호 (1, A, a,I, i)
    • reversed : 역순여부
      • 예) <ol type="A" start ="B" reversed>       <li>
  • <li> : 항목
