하이퍼 링크

  • 웹 페이지에서 다른 문서나 사이트로 이동할 수 있는 링크
  • <a> href="링크위치" target="실행방법"> 보여질 텍스트,이미지 </a>
    • 링크위치
      • 절대경로 : 서버의 루트위치를 기준으로 파일의 위치표시
        • 예) http://www.test-jsitclub.com/web/05/test.jpg
        •      /web/05/test.jpg
      • 상대경로 : 현재페이지의 위치를 기준으로 파일의 위치표시
        • ../ : 자신이 있는 폴더의 부모폴더
        •   ./ : 자신이 있는 폴더
      • 책갈피 : 링크에 #id를 이용하여 원하는 위치로 이동
    • target
      • _blank : 새 탭에 대상페이지를 실행
      • _self : 현재 창에서 실행
    • 하이퍼링크는 글 뿐아니라 이미지 및 여러태그에 한번에 적용할 수도 있다.

이미지

  • 웹 페이지에 이미지를 보여주는 태그
  • <img src="링크위치" alt="설명" >

오디오

  • 웹 페이지에서 음악파일을 재생하는 태그
  • <audio src="링크위치" controls="controls" >
  • 재생가능파일 : mp3,wav,ogg
  • 속성
    • controls="controls" : 컨트롤버튼
    • autoplay="autoplay" : 자동재생
    • loop="loop" : 반복재생
    • audio="muted" : 시작시 음소거

비디오

  • 웹 페이지에서 영상파일을 재생하는 태그
  • <video src="링크위치" >
  • 재생가능파일 : mp4,ogg,webm
  • 속성
    • controls="controls" : 컨트롤버튼
    • autoplay="autoplay" : 자동재생
    • loop="loop" : 반복재생
    • audio="muted" : 시작시 음소거

iframe

  • html 페이지 안에 다른 html 페이지를 보여주는 태그
  • 유튜브를 보여줄때도 <ifram>태그를 사용
    1. 유튜브검색 후 재생화면에서 오른쪽 클릭
    2. 메뉴에서 소스코드 복사
    3. 자신의 페이지 소스에 붙여 넣는다.
