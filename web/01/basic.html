HTML 기초

  • 정의 : HyperTextMarkupLanguage (하이퍼 텍스트 / 마크업 언어 )
  • 확장자 : html 또는 htm
  • 태그 : 태그를 통해 구분,효과나 기능을 적용한다. (태그표)
    • <태그>
    • <태그> 내용 </태그>
  • 주석 : <!--     -->
  • 기본 프레임
    • <!DOCTYPE html>
    <html>

    <head>

    <!-- 문서 정보를 넣는 부분 -->

    </head>

    <body>

    <!-- 문서 내용을 넣는 부분 -->

    </body>

    </html>

  • 공백 : &nbsp (특수기호)
  • 줄바꿈 : <br>
