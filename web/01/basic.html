HTML 기초

  • 정의 : HyperTextMarkupLanguage (하이퍼 텍스트 / 마크업 언어 )
  • 확장자가 html 이거나 htm 인 문서
  • 태그 : 태그를 통해 영역구분, 기능을 적용한다. (태그표)
    • < 태그 >
    • < 태그 > 내용 < /태그 >
  • 주석 : <!--     -->
  • 속성 : < 태그    속성="값" >의 형태 (전역속성)
  • 기본 프레임
    • <!DOCTYPE html>
    <html>

    <head>

    <!-- 문서 정보를 넣는 부분 -->

    </head>



    <body>

    <!-- 문서 내용을 넣는 부분 -->

    </body>


    </html>
  • 공백 : &nbsp (특수기호)
  • 줄바꿈 : <br>
