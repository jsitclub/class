CSS 기초 문법

  • style 바꾸는 방법
    • 태그의 속성 이용
      • <img src="aa.jpg" width="300">
    • style 속성 이용
      • <p style="color:purple">
    • style 태그 이용
      • 주로 < head> 부분에 작성
      • < style> div{ padding:10px; } < /style>
    • 외부 css파일 이용
      • link 태그를 이용하여 외부에 있는 css파일을 연결
      • <link rel="stylesheet" type="text/css" href="style.css">
    • 우선순위
      • style 속성 > style 태그 > 외부css파일 > 태그의 속성
      • id > class > 태그
  • class 속성 과 id 속성
    • 태그를 구분 하기위해 임의로 붙여준다.
    • class : 그룹 구분값
    • id : 유일한 구분값으로 페이지 안에서 겹치면 안됨
      • <div class="red" id="red01">
        <div class="red" id="red02">
    • style에서 class는 . 으로 id는 # 으로 구분한다.
      • .red{
             color : red;
        }

        #red01{
             padding : 10px;
        }
