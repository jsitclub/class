CSS 기초 문법

  • 속성:<태그 속성="값">의 형태
  • style 바꾸는 방법
    • 태그의 속성 이용
      • 예) <img src="aa.jpg" width="300">
    • style 속성 이용
      • 예) <p style="color:purple">
    • style 태그 이용
      • 예)
             div{
                  padding:10px;
              }
            < /style>
    • 외부 css파일 이용
      • 예) <link rel="stylesheet" type="text/css" href="style.css">
    • 우선순위
      • style 속성 > style 태그 > 외부css파일 > 태그의 속성
  • class 와 id
    • 예) <div class="red" id="red01">
         <div class="red" id="red02">
  • style에서 class는 . 으로 id는 # 으로 구분한다.
    • 예) .red{
              color:red;
         }

         #red01{
               padding:10px;
         }
