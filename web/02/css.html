CSS 기초 문법

  • 속성:<태그 속성="값">의 형태
  • style 바꾸는4가지 방법
    1. 태그의 속성 이용
      2. 예) <img src="aa.jpg" width="300">
    2. style 속성 이용
      3. 예) <p style="color:purple">
    3. style 태그 이용
      4. 예) <style>
             div{
                  padding:10px;
              }
            < /style>
    4. 외부 css파일 이용
      5. 예) <link rel="stylesheet" type="text/css" href="style.css">
  • class 와 id
    • 예) <div class="red" id="red01">
         <div class="red" id="red02">
  • style에서 class는 . 으로 id는 # 으로 구분한다.
    • 예) .red{
              color:red;
         }
         #red01{
               padding:10px;
         }
