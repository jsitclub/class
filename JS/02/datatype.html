데이터 타입

  • 원시타입
    • 숫자 : 4, -3, 2.5, 6.02e23
    • 문자 : 'hello', "jsitclub"
    • 논리 : true / false
    • 특수값 : Infinity,NaN
  • 객체타입 : 여러개의 변수를 모아 놓은 복합 데이터 타입
  • typeof : 자료형을 알려주는 명령 (ex) typeof 3

변수

  • 선언
    • var a;
    • var c,b=4;
    • 변수 선언의 끌어올림
  • 대입
    • a=3;
  • 사용
    • document.write(a+4);

연산자

  • 산술연산자
    • 산술 이항 연산자
      • +, -, *, /, %
    • 산술 대입 연산자
      • +=, -=, *=, /=, %=
    • 산술 단항 연산자
      • +a, -a, a++, ++a, a--, --a
  • 관계연산자
    • 관계연산자 : 결과를 논리값(True/False)로 반환.
      • ==(같다), !=(같지않다), <, <=, >, >=
      • ===(값과 타입이 모두같다), !==(값이나 타입이 같지않다)
      • 예)
        • var a=3, b='3';
        • console.log(a==b); //true
        • console.log(a!=b); //false
        • console.log(a===b); //false
        • console.log(a!==b); //true
  • 논리연산자
    • &&(and), ||(or), !(not)

형변환

  • 숫자로 변환 : Number(문자열)
  • 문자열로 변환 : String(숫자)
  • 논리값으로 변환 : Boolean() (주의!! Boolean([]) 는 true로 기본값이 무조건 false는 아님)
  • 문자열+숫자 => 문자열로 자동변환
