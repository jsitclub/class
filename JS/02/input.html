  • 웹 페이지에서 데이터를 입력을 받아 JavaScript를 사용하려면 HTML의 태그를 이용하여 정보나 이벤트를 입력 받은 후 JavaScript를 실행하는 방법을 주로 사용한다.
    그러므로 먼저 HTML의 입력 받을때 사용하는 태그를 먼저 알아보자.

< input >

  • < input >태그는 사용자에게 입력을 받을수 있는 태그로,type 속성에 의해 여러모양으로 사용 된다.
  • 속성
    • type : input 종류 설정 (다음표 참조)
    • name : 태그 이름 설정
    • value : 입력값
    • maxlength : 입력가능 길이
    • readOnly : 읽기전용
    • disabled : 활성화여부
    • checked : 선택된 값(체크박스,라디오버튼에서 사용)
        • 값 모양 설명
        사용안함 텍스트 입력
        text text 입력
        number 숫자입력
        password 패스워드 입력
        email 이메일 입력
        tel tel입력
        url url입력
        search search 입력
        button 버튼
        reset 폼의 입력값들을 초기화
        submit 폼의 입력값들을 전송
        checkbox 독서 코딩 운동 체크박스
        radio 자전거 도보 버스 라디오버튼(옵션버튼)
        range range 입력
        file 파일선택
        color 색선택
        date 날짜입력
        time 시간입력
        datetime-local 날짜/시간입력
        month 월입력
        week 주입력
        image submit 버튼의 이미지 정의
        hidden 숨겨진 값 처리

< button >

  • < button > 은 < input type="button"> 과 모양은 비슷하지만 마치는 태그인 < /button > 이 필요하고 사이값으로 텍스트나 이미지가 들어갈수 있다는 차이가 있다.
