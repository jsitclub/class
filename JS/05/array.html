배열

  • 배열 : 여러 값의 목록 (파이썬의 list(리스트)와 비슷)
  • 생성
    • var arr=[10,20,30,40];
    var arr1= new Array(100,200);
  • 읽기
    • console.log(arr[0]);
  • 추가 및 수정
    • arr.push(70); // 맨뒤에 추가
    arr[3]=100; // 중간에 수정
    arr.unshift(120) // 맨앞에 추가;
  • 삭제
    • arr.pop(); //맨뒤의 값을 삭제
    arr.splice(2, 1); //2번째 값을 지운다. 원래 이 함수는 splice(위치,갯수,바꿀값) 형태로 사용
    arr.shift(); //맨앞의 값을 삭제
  • 주의사항
    • // 1
    arr[10]=100;

    // 2
    delete arr[2];

    두가지 경우 모두 undefined 요소를 갖는 희소배열이 된다. (length속성을 사용할때 주의 할것!)
    hasOwnProperty(위치값)를 사용하면 undefined 여부를 확인할 수 있다. undefined일때 false
  • 속성, 메서드
    • length : 배열의 길이(속성)
    • concat(배열1,배열2) : 배열 합치기
    • slice(인덱스1,인덱스2) : 잘라오기
      • 매개변수 1개 일때 : 인덱스1 부터 끝 까지
      • 매개변수 2개 일때 : 인덱스1 부터 인덱스2미만 까지
    • toString() : 문자로 바꾸기
    • join(구분값) : 구분값으로 연결하기
